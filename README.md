# 🔗 URL Shortener

> A modern full-stack URL shortener built with React, Node.js, Express, MongoDB Atlas, Render, and Vercel.

---

# ✨ Features

✅ Shorten long URLs instantly

✅ Custom short links

✅ Links expire automatically after 7 days

✅ Protected reserved routes

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/qaxxiss/Url-Shortener.git
```

---

# 📦 Backend Setup

## Go to backend folder

```bash
cd backend
```

## Install dependencies

```bash
npm install
```

---

# 🎨 Frontend Setup

## Go to frontend folder

```bash
cd frontend
```

## Install dependencies

```bash
npm install
```

---

# 🔐 Environment Variables

Create a `.env` file inside:

```txt
backend/.env
```

Add:

```env
MONGO_URI=your_mongodb_connection_string
BASE_URL=your_custom_backend_link_or_api
PORT=5000
```

---

# ▶️ Run Locally

## Start Backend

```bash
cd backend
node server.js
```

---

## Start Frontend

```bash
cd frontend
npm run dev
```
